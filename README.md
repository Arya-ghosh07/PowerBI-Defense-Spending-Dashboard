# 🌍 Power BI Defense Spending Dashboard

## 📊 Overview
This dashboard analyzes global defense spending trends and compares USA with other countries.

## 🔍 Features
- Country vs USA comparison
- Spending trend analysis
- Global rank and share
- Radar chart comparison
- Waterfall (spending gap)
- AI Key Influencers

## 🛠 Tools
- Power BI
- DAX

## 📈 Insights
- USA dominates global defense spending
- Large gap between USA and other nations
- Increasing trend over time
